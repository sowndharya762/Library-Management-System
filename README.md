# Library Management System

## 📚 Project Overview
Library Management System is a software application used to manage library operations efficiently. It helps librarians maintain book records, issue books to students, return books, and track available books.

## 🚀 Features
- Add new books
- View all books
- Search books by ID or title
- Issue books to students
- Return books
- Update book details
- Delete book records
- Maintain book availability status

## 🛠 Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA / Eclipse


## 🗄 Database
Database Name: library_db

Tables:
- books
- students
- issued_books

## ▶ How to Run
1. Clone the repository.
2. Open the project in IntelliJ IDEA or Eclipse.
3. Configure MySQL database connection.
4. Run the SQL scripts to create tables.
5. Build the project using Maven.
6. Run the main application.

## 🎯 Objective
The main objective of this project is to automate library operations and reduce manual work while improving efficiency and accuracy.
